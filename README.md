# sandbox - Simple sand, fluid, and flame simulation

![img](sandbox.gif)

## Build/Run

#### Dependencies

- [raylib](https://github.com/raysan5/raylib)

#### Build

`make` to build, `./sandbox` to run

#### Controls

- `Left Mouse Button` - Add material
- `Right Mouse Button` - Remove material
- `L` - Load content 
- `S` - Save current content
